# Dependency Analysis Platform

## Description
Platform to analyze open-source dependencies and detect vulnerabilities automatically.

## Features (Milestone 2)
- Dependency file parsing
- Vulnerability scanning (pip-audit)
- JSON report generation
- CI pipeline with security checks

## Tech Stack
- Backend: Python (FastAPI)
- Scanner: pip-audit
- CI/CD: GitHub Actions

## How to Run
```bash
pip install -r requirements.txt
python main.py
